# Ejecutar en desarrollo

1. clonar el repositorio 
2. ejecutar
```
npm install
```
3. Tenete nest CLI install
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```

## Stack usado

* MongoDB
* Nest js
